About this tool

This tool can be used to visualize computations of the Curry programming language. It was created by Sascha Ecks in the context of a bachelor's thesis under Prof. Michael Hanus at the CAU Kiel.

Usage

Just enter a program and a main expression that should be evaluated into the corresponding fields. With the "term graph" option, the computation's term graphs are displayed as actual graphs, the "term graph as tree" displays the term graphs as trees, where shared nodes are displayed multiple times.

To avoid infinitely deep tree representations of term graphs in case of graphs with circles, a maximum depth for the generated tree-representations has to be specified if "term graph as tree" is selected.
Furthermore, the maximum number of execution steps is always limited to avoid infinite computations. The limit can be set manually, but cannot exceed {{max_steps}} steps.
